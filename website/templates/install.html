Install LocalWriter

Install the extension in LibreOffice, then point it at an OpenAI-compatible backend.

1. Install the extension

  1. Download the latest .oxt from the releases page.
  2. In LibreOffice, go to Tools > Extension Manager.
  3. Click Add and select the downloaded file.
  4. Restart LibreOffice.

2. Backend setup

LocalWriter needs an OpenAI-compatible API. Recommended options:

  • Ollama — Easiest for local usage.
  • text-generation-webui
  • OpenRouter / OpenAI — Cloud providers (API key required).

Settings

Configure everything under LocalWriter > Settings:

  • Chat/Text — Endpoint URL (e.g. http://localhost:11434 for Ollama), model, API key, temperature, context length, additional instructions, and the optional MCP Server (enable and port).
  • Image Settings — Size, aspect ratio, AI Horde options.

Connection keep-alive is on by default. For detailed examples (Ollama, OpenWebUI, OpenRouter, etc.), see CONFIG_EXAMPLES.md on GitHub.
